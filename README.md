# English Tutor AI

English Tutor AI — это интерактивное приложение для изучения английского языка с помощью искусственного интеллекта. Приложение позволяет пользователям практиковать разговорную речь, готовиться к собеседованиям, улучшать грамматику и навыки общения в различных сценариях.

---

## Основные функции

- **Свободное общение**  
  Практика речи на любые темы. Идеально подходит для улучшения разговорного английского.

- **Собеседование**  
  Подготовка к интервью на английском языке с моделями вопросов и обратной связью.

- **Путешествие: Аэропорт**  
  Практика прохождения регистрации и контроля в аэропорту на английском языке.

- **Разбор грамматики**  
  Подробные объяснения сложных правил английского с примерами.

---

## Особенности

- Реалистичный ИИ-репетитор уровня носителя языка.
- Встроенный текстовый чат с живым ответом ИИ.
- Возможность выбрать уровень английского (US/UK).
- Минималистичный интерфейс для сосредоточенной практики.
- Подходит для индивидуальных занятий и групповой практики.
